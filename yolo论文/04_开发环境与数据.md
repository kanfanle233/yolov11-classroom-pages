## 软件环境

### Python 环境

- Python 3.10+
- PyTorch 2.0+ (CUDA 11.8+)
- Ultralytics (YOLOv11)
- 其他依赖：见各子模块 requirements

### 关键模型权重

| 模型 | 文件 | 大小 | 用途 |
|------|------|------|------|
| YOLOv11n | `yolo11n.pt` | ~5MB | 轻量检测 |
| YOLOv11s | `yolo11s.pt` | ~18MB | 标准检测 |
| YOLOv11m | `yolo11m.pt` | ~50MB | 中等检测 |
| YOLOv11l | `yolo11l.pt` | ~83MB | 大型检测 |
| YOLOv11x | `yolo11x.pt` | ~110MB | 超大检测 |
| YOLOv11s-Pose | `yolo11s-pose.pt` | ~18MB | 姿态估计 |
| YOLOv11x-Pose | `yolo11x-pose.pt` | ~110MB | 高精度姿态估计 |

权重文件存放在根目录 `F:\PythonProject\pythonProject\YOLOv11\`。

## 数据集

### 智慧课堂学生行为数据集

- **位置**：`data/智慧课堂学生行为数据集/`
- **视角**：后方视角(23视频)、教师视角(28视频)、斜上方视角1(37视频)
- **案例**：正方视角案例（含标注JSON + 采样图片）

### SCB-Dataset3

- **位置**：`data/SCB-Dataset3 yolo dataset/`
- **规模**：0.355k 大学课堂YOLO标注数据
- **格式**：YOLO格式（images + labels, train/val split）

### 自制课堂检测数据集

- **位置**：`data/processed/classroom_yolo/`
- **用途**：自训练Case YOLO检测器（书本、笔、电子设备等课堂专属类别）

## 环境变量与路径

- 项目根目录：`F:\PythonProject\pythonProject\YOLOv11\`
- Python路径配置：`paths.py`
- 输出目录前缀：`output/` （流水线运行产物）

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 下载预训练权重（如需要）
# 权重文件已在根目录

# 3. 运行流水线（示例）
python scripts/main/09_run_pipeline.py --video data/xxx.mp4

# 4. 或使用集成入口
python integration/run_pipeline.py
```
